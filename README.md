Данный общий модуль добавляет функционал по взаимодействию с пользователем при работе с формами.

Если изменение каких-либо элементов формы влияет на видимость других заполненных элементов формы, то пользователю будет выводиться сообщение, что скрываемые реквизиты будут очищены. Пользователь может отказаться от изменений. В этом случае в изменяемом реквизите будет восстановлено его последнее значение.
